# /tmp/test-crud

Generated by [DRP](https://github.com/DarpanAdhikari/drp-go-cli) with JWT auth.

## Getting started

```bash
cp .env.example .env
# edit .env, especially JWT_SECRET and ENCRYPTION_KEY
drp migrate:seed
go mod tidy
drp run api
```

Auth routes:

- POST /auth/register
- POST /auth/login
- POST /auth/refresh
- POST /auth/logout
- GET /me
